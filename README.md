# Notion Widget Pomodoro

Notion に埋め込めるポモドーロタイマーのウィジェットです。ベースカラーに #40715a を使用し、ホワイトベースのカードデザインで集中・休憩サイクルを視覚化します。

## 主な機能
- 集中・短い休憩・長い休憩を切り替えるガラス調のウィジェット
- 円形のプログレスリングで経過時間を表示
- Start/Pause、Reset、Skip、Long Break ボタンを搭載
- 集中／休憩時間のカスタマイズに対応

## 使い方
1. 本リポジトリをクローンするか ZIP でダウンロードします。
2. `index.html` をブラウザで開くとタイマーが動作します。
3. Notion に埋め込む場合は、Vercel・Netlify・GitHub Pages など任意のホスティングに `index.html` を公開し、Notion の `/embed` ブロックに公開 URL を貼り付けてください。

## カスタマイズ
CSS のカラーパレットや JavaScript 内の `phases` 配列を変更すると、テーマカラーやサイクル構成を自由に調整できます。
